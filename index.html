16. HTML BLOCKs and InLine Elements

16.1 Block Level elements

A block level elements always starts on a new line.

The browser automatically add some space(a margin) before and after the element

A block-level element always takes up he full width available

two commonly used block elements are: p and div

the p tag is a block-level element

the div tag is a block-level element

16.2 Inline Elements

An inline element does not start on a new line.
An inline element only takes up as much width as necessary
Lets test the span element
This is a Span element

16.3 The div element

The div element is often used as a container for other HTML elements
The div element has no required attributes but Style , class and id are common
When used together with CSS, the div element can be used to Style blocks of content.

Lima

Lima is the capital of Peru.

16.4 The span element

The span element is an inline container used to mark up a part of a text or document.
When used together with the CSS, the span element can be used to style parts of the text

My mother has blue eyes and my father has dark green eyes.
